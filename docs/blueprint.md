# **App Name**: Membership Manager

## Core Features:

- Membership Enrollment: Streamlined process for enrolling new casino members.
- Membership Lookup: Quickly search for members by name, ID, or other criteria.
- Tier Management: Ability to view and update member tier levels.
- Reward Redemption: Interface for redeeming points and other rewards.
- Patron Profiling Tool: AI powered analysis tool to summarize key elements of customer visits based on transaction data.

## Style Guidelines:

- Primary color: Deep burgundy (#800020) evoking a sense of luxury and sophistication.
- Background color: Off-white (#F5F5DC) to provide a subtle contrast and a clean, neutral backdrop.
- Accent color: Gold (#FFD700) for highlights, important notifications, and calls to action.
- Headline font: 'Playfair', a modern serif that mirrors elements of Didot and expresses high fashion and a high-end feel. Body text: 'PT Sans', providing clean readability
- Use elegant, custom-designed icons that represent casino-related imagery, such as cards, chips, and rewards.
- Ensure a balanced and spacious layout with clear sections for each function to enhance user navigation.
- Incorporate subtle animations to enhance the user experience without being distracting.
